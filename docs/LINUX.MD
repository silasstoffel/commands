# General Commands

### while true

**Execute command with inity loop** 

```shell
 while true;do curl https://viacep.com.br/ws/29162206/json; echo; sleep 0.5; done
```


### watch

**Execute a command by specif time(seconds)** 

```shell
 watch -n5 curl https://viacep.com.br/ws/29162206/json 
```
